## 📦 Gerenciamento com Composer

Utilizo o **Composer** para gerenciar dependências PHP em todos os meus projetos, garantindo:

- **Autoloading** eficiente de classes
- **Versionamento** controlado de bibliotecas
- **Segurança** com atualizações monitoradas
- **Padronização** entre diferentes ambientes

### Principais Pacotes que Utilizo:

```json
{
    "require": {
        "vlucas/phpdotenv": "^5.6",
        "phpmailer/phpmailer": "^6.9",
        "google/apiclient": "^2.15"
    }
}
```

| Pacote | Finalidade |
|--------|-----------|
| **vlucas/phpdotenv** | Gerenciamento seguro de variáveis de ambiente (`.env`) |
| **phpmailer/phpmailer** | Envio de e-mails com SMTP (recuperação de senha, notificações) |
| **google/apiclient** | Integração com APIs do Google (Login Social, Google Drive) |
