## 🚀 Como Rodar Meus Projetos

```bash
# 1. Clone o repositório
git clone https://github.com/d4c137/NomeDoProjeto.git

# 2. Entre na pasta
cd NomeDoProjeto

# 3. Instale as dependências
composer install

# 4. Configure o ambiente
cp .env.example .env
# Edite o .env com suas credenciais

# 5. Inicie o servidor
php -S localhost:8000
```
