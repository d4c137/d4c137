## 📦 Gerenciamento com Composer

Utilizo **Composer** para gerenciar dependências PHP de forma profissional.

### Principais Pacotes:

```json
{
  "require": {
    "vlucas/phpdotenv": "^5.6",
    "phpmailer/phpmailer": "^6.9",
    "google/apiclient": "^2.15"
  }
}
```

| Pacote                    | Finalidade |
|---------------------------|----------|
| **vlucas/phpdotenv**      | Gerenciamento de variáveis de ambiente (`.env`) |
| **phpmailer/phpmailer**   | Envio de e-mails (recuperação de senha, notificações) |
| **google/apiclient**      | Integração com Google (Login Social) |
